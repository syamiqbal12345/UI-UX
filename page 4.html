    Tutorial Cara Install Django di Windows

    Pada kesempatan ini saya akan memberikan Tutorial Cara Install Django di Windows
    Untuk mengikuti tutorial kali ini sebaiknya baca dulu artikel tentang artikel Pengenalan Django Framework
    Agar lebih memahami tentang framework Django.

    Baik langsung saja kita mulai, untuk install Django silahkan step by step langkah dibawah ini

    1. Download dan install python (untuk download klik disini )
    2. Setelah di install, cek versi python yg menandakan bahwa python berhasil diinstal, dengan cara buka CMD dan ketik python --version Maka hasil nya akan seperti gambar dibawah

    3. Setelah install python,maka buat lah sebuah virtual environment, bagi yang belum mengenal virtual environment python bisa di baca diMengenal virtualenv di Python, lalu ketikkan perintah berikut:
      pip install virtualenvwrapper-win
      dan buat virtual environment nya dengan 
      mkvirtualenv nama_virtual_enviromentnya
      setelah menginstall virtual environment maka otomatis anda akan masuk ke virtual env yang sudah anda buat seperti contoh gambar dibawah.pada gambar dibawah saya membuat virtual env dengan nama tutorial



      Jika belum secara otomatis masuk ke virtual env dapat mengetikkan kode dibawah untuk masuk ke virtual environment
      workon nama_virtual_enviromentnya
    4. Yang terakhir adalah menginsall django, denga cara
      pip install django
      Setelah selesai mamasang django anda bisa mengeceknya dengan:
      django-admin --version
      Maka hasilnya akan seperti berikut

    5. Setelah selsai jalankan dengan visal studio code dengan perintah
      python manage.py runserver
      Maka hasilnya akan seperti ini
